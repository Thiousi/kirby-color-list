# Kirby Color List
A color picker in the form of a list of colors for Kirby

## Installation
Copy colorlist folder to your site/fields folder

## Usage
In  your blueprint:

  color:
    label: Color list
    type: colorlist
    options: 
      - "#eb1717"
      - "#28db78"
      - "#333333"
      - "#e74c3c"
      - "#353a62"
      - "#9b59b6"
      - "#d45353"
      - "#ffd041"
      - "#e91e63"
